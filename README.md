# 🌳 Tree Visualizer

An interactive web application to visualize and understand **Binary Tree traversals and reconstruction algorithms**.  
The tool demonstrates how different traversal techniques work and how binary trees can be reconstructed from traversal sequences.

🔗 Live Demo: https://tree-visualiser.netlify.app/

---

## 🚀 Features

- Visualize core **Binary Tree Traversals**
  - Inorder
  - Preorder
  - Postorder
  - Level Order (BFS)

- Construct binary trees using traversal combinations:
  - Preorder + Inorder
  - Inorder + Postorder
  - Preorder + Postorder
  - Level Order + Inorder

- Step-by-step **algorithm visualization** with dynamic tree rendering.

- Interactive interface to better understand **DFS and BFS traversal logic**.
